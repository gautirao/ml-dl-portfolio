# Category Definitions

Categories in LedgerMind Local help group transactions for better analysis.

## Common Categories
- **Groceries**: Food and household supplies from supermarkets.
- **Dining**: Restaurants, cafes, and takeaways.
- **Transport**: Public transport, fuel, and parking.
- **Bills**: Utilities, insurance, and regular service payments.
- **Entertainment**: Movies, streaming services, and leisure activities.
- **Uncategorised**: The default state for new transactions that haven't been assigned a specific category yet.

## Categorisation Process
1. **Suggestion**: When a new transaction is imported, the system suggests a category based on the merchant name and description.
2. **Approval**: Users must approve or change the suggested category in the UI.
3. **Learning**: The system prioritises user-defined rules for future transactions from the same merchant.
