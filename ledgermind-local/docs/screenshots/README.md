# Portfolio Screenshots Checklist

To effectively showcase LedgerMind Local, capture the following screenshots and place them in this directory.

## 1. Landing & Import
- [ ] **Empty State:** The dashboard before any data is imported.
- [ ] **Upload Preview:** The "Import" page showing the preview table after selecting `monzo_demo.csv`.
- [ ] **Success State:** The "Import" page after a successful upload.

## 2. Analytics Dashboard
- [ ] **Overview:** The main "Analytics" page showing spending summaries and charts.
- [ ] **Top Merchants:** The section showing where the most money is going.
- [ ] **Monthly Trends:** Any time-based charts showing inflow/outflow.

## 3. Conversational AI
- [ ] **Natural Language Query:** Asking a question like "What did I spend at Tesco?"
- [ ] **Planner & Evidence Panel:** The chat response with the "Planner & Evidence" section expanded, showing the SQL query and the tool results.
- [ ] **RAG Explanation:** Asking "How are recurring payments detected?" and showing the knowledge-based response with source citations.
- [ ] **Guardrail Refusal:** Asking "Should I invest in Nvidia?" and showing the polite refusal to give financial advice.

## 4. Human-in-the-Loop Categorization
- [ ] **Review Page:** The "Review Categories" page showing AI-suggested categories for new merchants.
- [ ] **Editing a Suggestion:** The modal or inline edit view for changing a suggested category.
- [ ] **Clean Data:** The transactions table showing perfectly categorized data after approval.

## 5. Semantic Search
- [ ] **Broad Query:** Searching for "transport" and seeing Trainline and Uber transactions matched semantically.
- [ ] **Index Management:** The search settings showing the "Rebuild Index" button and status.

---

**Tip:** Use a tool like [CleanShot X](https://getcleanshot.com/) or [Shottr](https://shottr.cc/) to capture clean, high-resolution screenshots with consistent padding.
